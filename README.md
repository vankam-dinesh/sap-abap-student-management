# Student Management System – SAP ABAP

## Description
A complete Student Management System developed using SAP ABAP.

## Features                                                                                                                                  
- Student Master Data Entry                                                             
- Marks Entry with Grade Calculation
-  Result Reporting using Join

## Technologies Used
- SAP ABAP                                                                                                                                           
- SE11 (Data Dictionary)
- SE38 (Executable Programs)                                                                                                                

## Programs
- ZSTUDENT_MASTER                                                                                         
- ZMARKS_ENTRY
- ZSTUDENT_REPORT

## Version Control
Designed to be version-controlled using abapGit.
